### 配置规则

如果您希望绕过某些网站或地区的代理（例如用于本地服务），您可以在 Clash 客户端的 **Rules（规则）** 选项卡中添加规则。示例规则：
- **DOMAIN-SUFFIX,baidu.com,DIRECT**
- **DOMAIN-SUFFIX,taobao.com,DIRECT**

这些规则确保流向特定域名的流量绕过代理。
